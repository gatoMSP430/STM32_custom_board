* Changes in ESATUtil 2.2.1, 2021-10-21

** Correction of memory handling errors.


* Changes in ESATUtil 2.2.0, 2021-02-10

** Streams are now seekable.

** Now buffers can use dynamic memory.

** There is a new module for scheduling tasks.

** There is a new module for handling CCSDS packet queues.

** ESAT_I2CSlave now can handle packets using packet queues.

** Flag containers now can read and write from/to streams.

** Minor error corrections.


* Changes in ESATUtil 2.1.2, 2019-01-10

** An error on how the module for handling boolean flags/bit vectors
reported the number of true flags has been fixed.


* Changes in ESATUtil 2.1.1, 2019-01-04

** Some portability issues have been fixed.


* Changes in ESATUtil 2.1.0, 2018-10-24

** Many errors have been fixed.

** All computer code files have copyright notices.

** There are example programs that illustrate the usage of the modules.

** There are new modules and functions for facilitating the work with
CCSDS Space Packets and KISS frames.

** There is a new module for computing CRC8 remainders.

** There is a new module for handling boolean flags/bit vectors.

** There is a new module for handling Semantic Version Numbers.

** There is a new clock interface.

** There are new encoding conversion functions.


* Changes in ESATUtil 2.0.0, 2017-11-08

** All existing modules have been rewritten.

** The old I2C and mathematics modules have been deleted.

** There is a new module for working with buffers with a stream interface.

** There are new modules for working with CCSDS Space Packets.

** There is a new module for working with KISS frames.

** There are new modules for passing CCSDS Space Packets through
the I2C interface.

** There are new modules for timekeeping.

** There are new type conversion functions.


* Changes in ESATUtil 1.0.0, 2017-02-09

** This is the initial release.
